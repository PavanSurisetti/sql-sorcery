## ⚙️ How to Use

1. Clone the repository:

   ```bash
   git clone https://github.com/PavanSurisetti/sql-sorcery.git
   ```

2. Open your preferred SQL environment:

   * MySQL Workbench
   * PostgreSQL
   * SQLite

3. Run the SQL files step by step to understand concepts progressively.

4. Modify queries and experiment to deepen your understanding.
